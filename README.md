# WCC Scanner — React Native App

A QR/barcode scanner mobile app built with React Native and deployed via Expo EAS.

## Features
- 📷 QR Code & Barcode Scanning
- 📱 Cross-platform (iOS & Android)
- 🚀 Production build via Expo EAS

## Tech Stack
- React Native
- Expo + EAS Build
- React Navigation

## Run Locally
```bash
npm install
npx expo start
```
